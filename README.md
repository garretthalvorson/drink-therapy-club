# Drink Therapy Club Landing Page

A super basic static landing page for Drink Therapy Club, intended for GitHub Pages and Etsy developer app verification.

## Local preview

Open `index.html` in a browser.

## GitHub Pages

Recommended deployment:

1. Create a public GitHub repo named `drink-therapy-club` or `drink-therapy-club-landing`.
2. Push this folder to the repo's `main` branch.
3. In GitHub, go to **Settings → Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

The URL will usually be:

`https://YOUR_GITHUB_USERNAME.github.io/REPO_NAME/`
